# Event-Management-System

---


## Prerequisites

---

* NodeJS
* Express
* nodemon
* cors
* ejs

```npm
npm install express nodemon cors ejs
```

##

---

<p> To start refer to npm command below</p>

```npm
npm run dev
```

or edit the package.json file and paste this

```json
{
  "name": "server",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "type": "module",
  "scripts": {
    "dev": "nodemon index.js"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "dependencies": {
    "cors": "^2.8.5",
    "ejs": "^3.1.9",
    "express": "^4.18.2",
    "mysql2": "^3.6.2"
  }

```
